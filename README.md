# 📝 Gestion-notes

**Application moderne de gestion de notes**  
Ce projet est une application moderne de gestion des notes, conçue pour offrir une expérience utilisateur fluide et efficace. Elle permet aux utilisateurs de créer, organiser, modifier et sécuriser leurs notes de manière intuitive, avec une interface moderne et responsive adaptée à tous les appareils.

## 🧠 Fonctionnalités principales

- 🆕 Création, édition et suppression de notes  
- 📚 Organisation des notes par catégories ou tags  
- 🔐 Authentification des utilisateurs  
- 🖥️ Interface web rapide et responsive  
- ⚡ Performance optimisée grâce à **Vite** et un bundling moderne
